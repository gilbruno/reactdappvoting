# reactdappvoting

## Description

Dapp using the smart contract _contracts/Voting.sol_

## Install dependencies

Run : 

```
npm install
```

in the _root_ directory and the _client_ directory

## Launch

Go to the _client_ directory and run :

```
npm run start
```
